# Depelos Fan Page

Página fan dedicada a Depelos. Aquí encontrarás noticias, enlaces, imágenes y contenido creado por y para la comunidad.

## Demo
La web está publicada con GitHub Pages en:
https://adrischz.github.io/Depelosfanpage.github.io/  (ajusta la URL si usas un dominio distinto)

## Qué hay en este repositorio
- Archivos estáticos de la web (HTML, CSS, JS, imágenes).
- Contenido y recursos para la página fan de Depelos.

## Cómo ver el sitio localmente
Opción 1 — servir archivos estáticos con Python 3:
1. Clona el repositorio:
   git clone https://github.com/adrischz/Depelosfanpage.github.io.git
2. Entra en la carpeta del proyecto:
   cd Depelosfanpage.github.io
3. Inicia un servidor local:
   python3 -m http.server 8000
4. Abre http://localhost:8000 en tu navegador.

Opción 2 — usar una extensión de servidor (VS Code Live Server) o cualquier servidor estático.

(Si el sitio está hecho con Jekyll u otro generador, indícalo y actualizo estas instrucciones.)

## Cómo contribuir
- Abre un issue para proponer cambios o reportar problemas.
- Para cambios rápidos, crea una rama nueva, haz commits y envía un pull request.
- Si quieres añadir contenido (artículos, imágenes, secciones), crea una carpeta `content/` o sigue la estructura existente y documenta el formato.

Pequeña guía:
1. Fork del repositorio.
2. Crea una rama: `git checkout -b feature/nueva-seccion`
3. Realiza los cambios y haz commits claros.
4. Abre un Pull Request describiendo los cambios.
